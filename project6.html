About the Game

The **Cart Racing Game** is a JavaScript-based racing game where players control a cart across different tracks. Download the game from my **GitHub repository** and follow the instructions in the README to set it up and play!

👉 Visit the GitHub Repo & Play the Game

Gameplay Video

Watch a recorded gameplay session of the Cart Racing Game.

Game Features

  • 🚗 Physics-based movement with acceleration and braking.
  • 🏁 Different track layouts and difficulty levels.
  • 🎨 Customizable cart colors and styles.
  • 🕹️ Simple controls (Arrow keys to drive).

How to Play

To play the game, follow these steps:

  1. Go to the GitHub Repository.
  2. Clone the repository using: 
    git clone https://github.com/moroianu13/cart-racing-game.git
  3. Navigate into the project folder: 
    cd cart-racing-game
  4. Install dependencies (if any): 
    npm install
  5. Run the game: 
    npm start

Technical Stack

  • 🟢 **JavaScript** - Game logic and physics.
  • 🎨 **CSS** - UI styling and animations.
  • 📁 **HTML5 Canvas** - Rendering game graphics.
  • 🚀 **GitHub** - Version control & hosting.
