Overview

Berlin RentWise is a production-ready Django web application deployed on Fly.io that helps users explore 447 Berlin neighborhoods across 12 boroughs. The platform combines interactive data visualization, REST API endpoints, and an AI-powered chatbot using Google Gemini to provide comprehensive insights into rental prices, crime statistics, amenities, and lifestyle preferences. With automated CI/CD pipelines and a robust PostgreSQL backend, this project demonstrates full-stack development expertise from data processing to cloud deployment.

System Design

The application follows a modern CI/CD workflow:

  1. GitHub Repo ‚Üí Push to main branch
  2. GitHub Actions ‚Üí Run Django CI tests with PostgreSQL
  3. Fly Deploy ‚Üí Automated deployment (if tests pass)
  4. Fly.io Production ‚Üí Gunicorn, PostgreSQL, Auto-scaling
  5. External APIs ‚Üí Gemini AI, Weather API, Wikipedia

Application Flow

  1. User Request ‚Üí Nginx/Fly.io proxy ‚Üí Gunicorn
  2. Django Views ‚Üí Process request ‚Üí Query PostgreSQL
  3. Chatbot Request ‚Üí Dialog matching ‚Üí Predefined responses ‚Üí AI API
  4. API Calls ‚Üí DRF ViewSets ‚Üí Serializers ‚Üí JSON response
  5. Static Files ‚Üí WhiteNoise ‚Üí Cached & compressed

üìö API Documentation

REST Endpoints

Neighborhoods

  • GET /api/neighborhoods/ - List all neighborhoods
  • POST /api/neighborhoods/ - Create neighborhood
  • GET /api/neighborhoods/{id}/ - Retrieve neighborhood
  • PUT /api/neighborhoods/{id}/ - Update neighborhood
  • DELETE /api/neighborhoods/{id}/ - Delete neighborhood

Boroughs

  • GET /api/boroughs/ - List all boroughs
  • POST /api/boroughs/ - Create borough
  • GET /api/boroughs/{id}/ - Retrieve borough
  • PUT /api/boroughs/{id}/ - Update borough
  • DELETE /api/boroughs/{id}/ - Delete borough

Chat

  • POST /chat/ - Send chat message
  • Body: {"message": "Your question here"}
  • Response: {"response": "AI/bot response"}
